# README #

Download ERA Interim data:
- temperature 2m, 
- total cloud cover
- wind speed
- wind direction
- geopotential height
- mean sea level pressure 

Convert from GRIB to netCDF format.

Edit config template and save as a new file.

Edit T_downscaling.R and provide the path and filename of your config file.

Run the code.
